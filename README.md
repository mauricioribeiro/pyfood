# pyfood
pyFood is a Django web application connected with chatbot (via Dialogflow)

### Tecnologias utilizadas

#### Back-End:
 - Python 3.4
 - Django
 - RESTFul API
 - Dialogflow

##### Front-End
 - AngularJS
 - Material Design
 - Npm e Bower
